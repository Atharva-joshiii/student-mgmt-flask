# Student Management System

## Description
A full-stack web app for managing students in a university context.
Built with Flask, MongoDB, and Bootstrap 5.

## Features
- Student CRUD (add, edit, delete)
- Attendance tracking per subject with percentage calculation
- Marks entry and management
- Complaints portal with remarks
- User authentication (register/login/logout)
- Profile picture upload
- CSV export

## Tech Stack
- Backend: Python, Flask, Flask-Login, bcrypt
- Database: MongoDB (local + Atlas)
- Frontend: Bootstrap 5, Jinja2 templates

## How to Run
pip install -r requirements.txt
python app.py
